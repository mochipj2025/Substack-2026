# ケツ診断 Deep Research 整理メモ

作成日: 2026-06-23

## 目的

Deep Research の5本を、ケツ印どうぶつ診断の実装に使える形へ整理する。
既存占いの本文やブランドをそのまま使うのではなく、研究メモを参考資料として扱い、公開する診断文は「ケツ診断」オリジナルにする。

## 読み込んだ資料

| ファイル | 主題 | ケツ診断での扱い |
|---|---|---|
| `deep-research-report.md` | 五行診断 | 質問紙、スコア、状態分類の中核に使う |
| `deep-research-report (1).md` | 星座占い・西洋占星術 | 太陽星座のみ軽い感性補正として使う |
| `deep-research-report (2).md` | 血液型占い | 文化的な会話フック、対人距離感の軽い補正に留める |
| `deep-research-report (3).md` | 動物占いロジック | 生年月日ベースの12動物、60分類、相性リングの参考にする |
| `deep-research-report (4).md` | 数秘術ロジック | 生年月日のライフパスを人生テーマとして使う |

## 全体方針

ケツ診断は、既存占いの正統性を競うサイトではなく、ケツ印一家で楽しく見せ合えるオリジナル診断にする。

そのため、5つの要素は次のように役割分担する。

| 要素 | ケツ診断での役割 | 表示上の言い方 |
|---|---|---|
| 動物 | 本質キャラ、反応パターン | 「あなたの本体キャラ」 |
| 五行 | 今のエネルギーの出方、行動傾向 | 「いま出やすい属性」 |
| 数秘 | 人生テーマ、伸びる方向 | 「奥にあるテーマ」 |
| 星座 | 感性、世界の見方 | 「空気の感じ取り方」 |
| 血液型 | 対人距離感、雑談フック | 「人との距離の取り方」 |

## 重要な判断

### 1. 医療・科学っぽく断定しない

五行レポートは、五行診断が歴史的には深い一方で、現代科学の基準では診断者差や妥当性の弱さがあると整理している。

ケツ診断では、五行を医療診断や体質診断として出さない。
使うのは「行動パターン」「エネルギーの向き」「今の状態」の分類。

### 2. 星座は本格ホロスコープではなく太陽星座に留める

星座レポートでは、出生時刻・出生地がないとアセンダント、ハウス、月、トランジットの精度が不安定になると整理されている。

今のアプリでは出生時刻・出生地を取っていないので、星座は太陽星座のみ。
ホロスコープ風チャートを作る場合も「本格占星術」ではなく「ケツ印チャート」「どうぶつ星盤」として扱う。

### 3. 血液型は低ウェイトにする

血液型レポートでは、性格との強い関連は主流の証拠では支持されず、ラベリングや差別リスクが重要だと整理されている。

したがって、血液型は診断の主判定に使わない。
結果文では、対人距離感や会話の味付けとして軽く扱う。

避ける表現:

- 「A型だから几帳面」
- 「B型だから自己中心的」
- 「この血液型とは相性が悪い」

使う表現:

- 「人との距離を整えたい時に出やすいクセ」
- 「周囲との関わり方の補助メモ」
- 「診断の味付け」

### 4. 動物占いは商標・公式互換に注意する

動物占いレポートでは、公式系統が複数あり、現行精密版には節入日や23時ルールなどの未公開要素があると整理されている。

ケツ診断では、次のように分ける。

- 内部ロジック: 生年月日ベースの60分類互換ロジックを参考にする
- 表示名: 「ケツ印どうぶつ」「本質どうぶつ」など独自名称にする
- 本文: 公式文章や図表を転載せず、ケツ印文体で再構成する
- 注意: 「公式完全互換」とは言わない

### 5. 数秘はまずライフパスだけ使う

数秘術レポートでは、姓名数秘は翻字、Yの扱い、長音表記などで結果が変わりやすいと整理されている。

今のアプリでは、生年月日だけで計算できるライフパスを使うのが安定。
姓名数秘は後回しにする。

実装仕様:

- 生年月日からライフパスを算出
- マスターナンバーは `11 / 22 / 33` を保持
- 表示では「人生テーマ」「伸びる方向」として扱う

## ケツ診断の内部データ設計

診断結果は、最初に5要素をそのまま出すのではなく、内部ステータスへ変換する。

```js
const diagnosis = {
  animal: "tanuki",
  characterNumber: 41,
  element: "earth",
  subElement: "metal",
  numerology: 9,
  zodiac: "taurus",
  bloodType: "O"
};
```

これをケツ診断ステータスへ変換する。

```js
const status = {
  intuition: 3,
  action: 2,
  stability: 5,
  social: 4,
  creativity: 3,
  recovery: 5
};
```

## レーダーチャート軸

最初の実装は6軸が扱いやすい。

| 軸 | 意味 | 主に影響する要素 |
|---|---|---|
| 直感 | 感じ取る力、空気を読む力 | 水、星座、動物 |
| 行動 | 始める力、突破力 | 火、数秘、動物 |
| 安定 | 続ける力、守る力 | 土、血液型、動物 |
| 社交 | つなぐ力、場づくり | 木、血液型、動物 |
| 創造 | 表現、発想、遊び心 | 木、火、数秘、星座 |
| 回復 | 整える力、戻る力 | 土、水、五行サブ属性 |

## スコア合成案

最初はシンプルな加点式でよい。

1. 動物ごとのベースステータスを置く
2. 五行メインで2点加点
3. 五行サブで1点加点
4. 数秘で1〜2軸を補正
5. 星座で感性系を軽く補正
6. 血液型で対人系を軽く補正
7. 最後に1〜5へ丸める

血液型は低ウェイトにする。
血液型だけでステータスが大きく変わらないようにする。

## 星座のケツ診断化

星座は、感性の季節感として扱う。

- 火の星座: 表現、勢い、自己発火
- 地の星座: 現実感、継続、手触り
- 風の星座: 言葉、情報、関係性
- 水の星座: 感情、共感、記憶

この分類は、結果文の口調やレーダーチャートの軽い補正に使う。
出生時刻・出生地なしで本格ホロスコープとは言わない。

## 血液型のケツ診断化

血液型は「当てる」より「話しやすくする」ための補助軸。

| 血液型 | ケツ診断での扱い |
|---|---|
| A | 整えて安心したい時のクセ |
| B | 自分のリズムを大事にしたい時のクセ |
| O | 大きく受け止めて動きたい時のクセ |
| AB | 距離を取りながら全体を見たい時のクセ |
| 不明 | 血液型に寄せず、他要素をそのまま見る |

## 今日の運勢ロジック案

運勢はランダムではなく、日付と本人の診断結果から決める。

```js
const dailyNumber = reduceDate(today);
const dailyElement = elementFromDate(today);
const relation = fiveElementRelation(profile.element, dailyElement);
const seed = dailyNumber + animalIndex + elementIndex + numerology;
```

出力する運勢:

- 総合運
- 人間関係運
- 活動・仕事運
- 創作運
- 回復運

五行関係で文脈を作る。

| 関係 | 読み方 |
|---|---|
| 今日の五行が本人を生む | 追い風の日 |
| 本人が今日の五行を生む | 使うほど成果が出るが消耗注意 |
| 同じ五行 | 自分らしさが強く出る日 |
| 今日の五行が本人を剋す | 調整、学び、予定変更の日 |
| 本人が今日の五行を剋す | 主導権を握れるが強引注意 |

## ホロスコープ風チャート案

本格占星術ではなく、ケツ診断の可視化として作る。

名称案:

- ケツ印チャート
- どうぶつ星盤
- ケツ印ステータス盤

表示案:

- 中央: 動物アイコン
- 外周: 五行リング
- 内側: 6軸レーダーチャート
- 周囲: 数秘、星座、血液型、今日の五行
- 下部: ケツ印称号と一言

## 実装優先順位

### Phase 1: 設計を固定

- 6軸ステータスを確定する
- 動物12種のベースステータスを作る
- 五行、数秘、星座、血液型の補正表を作る
- 診断結果から `status` を生成する関数を作る

### Phase 2: 結果ページへ反映

- 結果カードに上位3ステータスを出す
- レーダーチャートを追加する
- 称号を「動物 × 五行 × 上位ステータス」から生成する
- 総合コメントをステータスベースにする

### Phase 3: 共有体験へ展開

- 4:5結果カードPromptにステータスを入れる
- SNSアイコンPromptに称号と動物を入れる
- PDFにレーダーチャートを入れる
- 今日の運勢を追加する

## 結論

Deep Research の内容をそのまま「正しい占い」として採用するより、次の使い方が一番強い。

- 動物: 覚えやすい本質キャラ
- 五行: スコア化できる行動エネルギー
- 数秘: 生年月日から出る人生テーマ
- 星座: 感性の季節感
- 血液型: 雑談しやすい対人補助

そして最終表示は、既存占い名ではなく「ケツ診断」として出す。
これなら、レーダーチャート、今日の運勢、相性診断、PDF、画像Prompt、Pet化まで自然に派生できる。
