# ケツ印どうぶつ診断 結果ページ構成案

## 目的

診断結果ページは、本人が「自分の結果」として読めることを最優先にする。
素材確認や他タイプ閲覧は混ぜず、ほかの結果を見たい場合は診断しなおす導線にする。

## 表示する結果の要素

1. 動物タイプ
   - 既存の動物占い12タイプをベースにする
   - 公開文はケツ印一家向けのオリジナル文章にする
   - 例：ライオン、狼、こじか、猿、チータ、黒ひょう、虎、たぬき、子守熊、ゾウ、ひつじ、ペガサス

2. 五行タイプ
   - 質問紙で判定する
   - 木・火・土・金・水
   - 「今の行動パターン」「エネルギーの出方」として扱う

3. 数秘術
   - 生年月日からライフパスナンバーを出す
   - 1-9、11、22、33に対応
   - 「人生テーマ」「才能の使い方」として扱う

4. 血液型
   - A、B、O、AB、不明
   - メイン判定ではなく、人付き合いのクセとして扱う

## ページ構成

### 1. 結果ヒーロー

表示内容：

- あなたは「火のライオン」
- 動物画像
- ケツ印称号
- 一言サマリー
- 診断しなおすボタン

目的：

最初の画面で「自分の結果が出た」とわかるようにする。

### 2. 基本性格

表示内容：

- 動物タイプの基本性格
- 強み
- 苦手になりやすいこと
- 周りから見た印象

目的：

既存の動物占いらしい納得感を作る。
ただし文章は転載せず、要点をケツ印文体で再構成する。

### 3. 詳しい分析部屋

3カラムまたは縦並びで表示：

- 五行分析
- 数秘術分析
- 血液型分析

目的：

結果に深みを出し、動物タイプだけでは足りない個人差を表現する。

### 4. ケツ印流の読み解き

表示内容：

- 動物 × 五行
- 動物 × 数秘
- 動物 × 血液型
- 総合コメント

例：

ライオン × 火は、場を仕切るだけでなく空気を温める親分タイプ。
数秘8が入ると、責任や成果への意識が強くなる。
O型なら、頼られるほど本気が出る。

### 5. 人間関係

表示内容：

- 近づきやすい相手
- ぶつかりやすい場面
- 仲良くなるコツ

目的：

メンバー同士で見せ合った時に会話が生まれるようにする。

### 6. 今日の使い方

表示内容：

- 今日意識するといいこと
- 疲れた時の戻り方
- ケツ印ひとこと

目的：

読んで終わりではなく、日常に持ち帰れる形にする。

## データ設計案

結果は次のデータを合成して作る。

```json
{
  "animal": "lion",
  "element": "fire",
  "numerology": 8,
  "bloodType": "O"
}
```

### 読み込みデータ

- `animal-readings.json`
  - 動物12タイプの基本結果
- `element-readings.json`
  - 五行5タイプの分析
- `numerology-readings.json`
  - 数秘1-9、11、22、33
- `blood-readings.json`
  - 血液型5タイプ
- `combination-readings.json`
  - 動物 × 五行などの組み合わせコメント

## 注意点

- 既存の動物占い本文は転載しない
- 「動物占い」名称の扱いは公開前に確認する
- 診断の根拠として使う表やロジックも、出典と利用可否を確認する
- 内輪公開でも、結果文はオリジナルにする

## 最初に作るべき結果データ

優先順位：

1. 動物12タイプの基本結果
2. 五行5タイプの分析
3. 数秘12パターン
4. 血液型5パターン
5. 組み合わせコメント

最初の完成形は、12動物 × 5五行 = 60通りの称号と一言コメントを出せる状態。
