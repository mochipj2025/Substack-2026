# ケツ印ジョブ 60通り称号案

作成日: 2026-06-24

## 目的

五行5種 × 動物12種の第一候補セットから、結果トップに出せる称号を60通り一覧化する。
ここでは実装前の確認用として、読みやすさを優先した表示名を置く。

## 命名ルール

基本形:

```text
{五行冠}{動物ジョブ}
```

ただし、動物ジョブに「の」が多い場合は短縮する。

例:

```text
翠の + 夜道の案内人 -> 翠の夜道案内人
紅蓮の + 疾風の開拓者 -> 紅蓮の疾風開拓者
大地の + 相談役 -> 大地の相談役
```

## 次の実装メモ

`animals.js` に入れる場合は、以下のようなデータを作る。

```js
const ketsuJobTitles = {
  wood: {
    wolf: "翠の夜道案内人",
    deer: "翠の心番人",
    monkey: "翠の閃き奇術師"
  }
};
```

K-RPGコードは称号とは別で作る。

```js
const ketsuCode = buildKetsuRpgCode({
  element,
  animal,
  numerology,
  zodiac
});
```
