# ケツ印ファンタジージョブ命名案

作成日: 2026-06-24

## 目的

ケツ診断の結果トップに出す「覚えやすい称号」を作る。
MBTIのように共有しやすく、かつケツ印らしくキャラクター化できる名前にする。

## 基本方針

- かわいすぎより、ファンタジー寄り
- ただし初見で意味がわかる
- 「何をする人か」が伝わる
- 公式占いの表現をそのまま使わず、ケツ診断オリジナルにする
- 診断結果は「称号 + 型コード + 5要素タグ」で表示する

表示例:

```text
あなたのケツ印ジョブ
大地の相談役

土のたぬき / 数秘9 / 牡牛座 / O型
K-RPG: ESCS
```

## 組み合わせルール

### ルールA: 短縮結合

五行冠と動物ジョブをそのまま足すと「の」が重くなるため、動物ジョブ側を短くする。

```text
紅蓮の + 先陣の突撃兵
= 紅蓮の突撃兵
```

### ルールB: 属性エンチャント

五行を1語の属性として、ジョブに付ける。

```text
翠 + 心の番人 = 翠の心番人
鋼 + 美学の騎士 = 鋼の美学騎士
```

### ルールC: 叙事詩スタイル

五行冠、動物のコア単語、ジョブの役割を組み合わせる。

```text
月影 + 星 + 旅人 = 月影の星旅人
豊穣 + 智慧 + 相談役 = 豊穣の智慧相談役
```

## K-RPGコード案

MBTIのように短く共有できるコードを用意する。

```text
紅蓮の先陣開拓者（FAII型）
```

### 1文字目: 五行

| 五行 | コード |
|---|---|
| 木 | W |
| 火 | F |
| 土 | E |
| 金 | M |
| 水 | A |

注意:
水の `A` は Aqua。
土の `E` は Earth。
金の `M` は Metal。

### 2文字目: 動物クラス

| クラス | コード | 動物 |
|---|---|---|
| Attacker | A | チータ、ライオン、狼 |
| Defender | D | 虎、ゾウ、黒ひょう |
| Support | S | こじか、ひつじ、たぬき |
| Caster | C | ペガサス、猿、子守熊 |

### 3文字目: 数秘ベクトル

| ベクトル | コード | 数秘 |
|---|---|---|
| Individual | I | 1, 3, 5, 7, 9, 11, 33 |
| Cooperate | C | 2, 4, 6, 8, 22 |

### 4文字目: 星座3区分

| 区分 | コード | 星座 |
|---|---|---|
| Initiative | I | 牡羊、蟹、天秤、山羊 |
| Stable | S | 牡牛、獅子、蠍、水瓶 |
| Flexible | F | 双子、乙女、射手、魚 |

## K-RPGコードの改善メモ

この案はかなり良いが、コードの読みやすさには改善余地がある。

課題:

- `A` が水とAttackerの両方に見える
- `M` がMetalで少し直感から遠い
- `I` がIndividualとInitiativeで重なる
- 日本語ユーザーには英字コードだけだと少し説明が必要

改善案:

### 案A: 英字コードを維持

MBTI風にしたいなら採用。
説明欄に「K-RPGコード」として表示する。

### 案B: 日本語タグを併記

```text
FAII型
火 / 攻 / 個 / 先
```

これが一番わかりやすい。

### 案C: 結果トップは称号、コードはサブ表示

```text
紅蓮の先陣開拓者
FAII型 - 火 / 攻 / 個 / 先
```

最初の実装は案Cがよい。

## 実装時の優先順位

1. `buildKetsuJobTitle(profile)` を作る
2. `buildKetsuRpgCode(profile)` を作る
3. 結果ページ上部にジョブ名を表示する
4. 詳細分析のタグにK-RPGコードを出す
5. Image PromptとPDFにもジョブ名を入れる
