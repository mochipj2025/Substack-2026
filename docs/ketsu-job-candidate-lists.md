## 5. 血液型補助候補

血液型は主判定にしない。
対人距離感の軽い補助として扱う。

| 血液型 | 補助イメージ | 表示に使える言葉 |
|---|---|---|
| A | 整える、丁寧に進める | 整え / 丁寧 / 守り / 余白 |
| B | 自分のリズムを守る | 自由 / 旅 / 独自 / 気まま |
| O | 大きく受け止めて動く | 大らか / 旗振り / 包容 / 前進 |
| AB | 距離を取りながら全体を見る | 観測 / 二面性 / 影 / 星図 |
| 不明 | 血液型に寄せない | 未設定 / 余白 / 謎 / 自由枠 |

## 6. K-RPGコード候補

### 4文字コード

```text
五行 + 動物クラス + 数秘ベクトル + 星座区分
```

例:

```text
ESCS型 = 土 / 支援 / 協調 / 安定
FAII型 = 火 / 攻撃 / 個性 / 先駆
```

### 1文字目: 五行

| 五行 | 英字 | 日本語タグ |
|---|---|---|
| 木 | W | 木 |
| 火 | F | 火 |
| 土 | E | 土 |
| 金 | M | 金 |
| 水 | A | 水 |

### 2文字目: 動物クラス

| クラス | 英字 | 日本語タグ | 動物 |
|---|---|---|---|
| Attacker | A | 攻 | チータ、ライオン、狼 |
| Defender | D | 守 | 虎、ゾウ、黒ひょう |
| Support | S | 援 | こじか、ひつじ、たぬき |
| Caster | C | 術 | ペガサス、猿、子守熊 |

### 3文字目: 数秘ベクトル

| ベクトル | 英字 | 日本語タグ | 数秘 |
|---|---|---|---|
